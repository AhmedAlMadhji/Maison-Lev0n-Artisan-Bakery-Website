# Maison Lev0n — Artisan Bakery Website

Premium artisan bakery brand website built with HTML, CSS, and Vanilla JavaScript.

**Live demo:** [AhmedAlMadhji.github.io/Maison-Lev0n-Artisan-Bakery-Website](https://ahmedalmadhji.github.io/Maison-Lev0n-Artisan-Bakery-Website/)

## Features

- Editorial layout with 10 content sections + portfolio case study
- Mobile navigation, sticky header, active section highlighting
- Product category filtering
- Testimonial slider with autoplay
- Order form with client-side validation
- Scroll reveal animations (respects `prefers-reduced-motion`)
- SEO meta tags, Open Graph, JSON-LD structured data

## Run Locally

Open `index.html` in a browser, or use a local server:

```bash
npx serve .
```
